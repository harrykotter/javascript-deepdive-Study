---

- [미리] 매주 정해진 챕터 공부하고 내용 정리하기

- [미리] 질문 만들어오기 (면접 질문 또는 문제)

- 랜덤으로 각 챕터마다 발표자 선정하여 발표하기

- 공부하면서 궁금했던 내용 질문 및 답변 시간

- 준비한 질문 공유 및 답변

- 스터디 당일 12:00 이전까지 PR을 날려야 합니다.(브랜치 생성 후 자신의 브랜치로)
